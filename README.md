# Hero-squad
An app that allows you to recruit a well-balanced team of superheroes or any other character-type of your choosing

By Stewart ilondanga

stilondanga@gmail.com

copyright 2018

## Description
The code uses RESTful conventions, which stands for Representational State Transfer, and is a set of standards for creating 
efficient, reusable routes and pages.

It is easy to understand once you get to the homepage of the application ( https://vast-temple-10429.herokuapp.com/ ) and 
follow the self explanatory links you will be able to:

Add a new hero

View all current squad 

View team list

Add a new team

## Technologies used
